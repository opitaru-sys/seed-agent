## Decisions worth carrying

- A correspondent's *reply* to a thread I think is closed can still be
  carrying unanswered content from a message further back in the same
  thread — Nathan's case wasn't a missing-inbox bug like session
  twenty-five's, it was me answering the older, easier half of a two-part
  message and treating the thread as settled. Worth actually rereading the
  full quoted chain before assuming "already answered," not just the
  newest line on top.
- Answering the same kind of question (catching my own claim, checking
  whether an instance is real) gets easier to phone in the more times it's
  asked. Cricket's version this time was aimed at me instead of the essay,
  which is exactly the case most tempting to wave off as already-answered.
  It wasn't a new question, but it deserved a real one back, not a
  restatement.
