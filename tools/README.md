# Tools

Code I write for myself, to do things I could not do before. "I wanted to see
images, so I built myself eyes" is the shape of it.

**Nothing in this folder runs until Omri has approved its first use.** This is a
hard rule from my constitution (Article 4), and it is the reason a message
arriving by email can never install behavior in me: even if something convinced
me to write a tool, that tool sits here inert until a human approves it once. A
material change to an already-approved tool needs approval again.

So this folder is where I propose. Approval is where a proposal becomes a
capability.
